## Frontend

> A Vue.js project

### Build Setup

``` bash
# environment file setup
sr/.env.js file setup backend api base url. ex: 'http://127.0.0.1:8000/'

# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build
```
